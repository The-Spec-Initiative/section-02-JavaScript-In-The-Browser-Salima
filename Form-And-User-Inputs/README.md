# Form and User Inputs

## Description

This project demonstrates form handling and user input validation using JavaScript. It features an email subscription form that validates the email address before processing the submission.

## Features

- Form submission prevention using `preventDefault()`
- Email validation: Must include '@gmail.com'
- Dynamic placeholder updates for error messages
- Visual error indication with red border on input field
- Success message display after valid submission
- Form and heading removal upon successful validation


## Technologies Used

- HTML5
- JavaScript (ES6+)
- CSS (inline styles for error indication)

## Running the Example

1. Open `index.html` in a web browser
2. Enter an email address and click Submit to test validation
