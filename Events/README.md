# Events and Form Validation

## Description

This project demonstrates event handling and form validation using JavaScript. It includes a registration form that validates user input for name, email, and password before submission.

## Features

- Form submission prevention using `preventDefault()`
- Input validation for:
  - Name: Cannot be empty
  - Email: Must include '@gmail.com'
  - Password: Must be at least 8 characters long
- Dynamic placeholder updates for error messages
- Success message display after valid submission
- Form and heading removal upon successful validation


## Technologies Used

- HTML5
- JavaScript (ES6+)

## Running the Example

1. Open `index.html` in a web browser
2. Fill out the form and click Submit to test validation
