# DOM Manipulation Example

This project demonstrates basic DOM manipulation using JavaScript in the browser. It showcases how to select elements, add event listeners, and modify the content and structure of a webpage dynamically.

## Features

- Interactive button that responds to user clicks
- Dynamic text update using `innerText`
- Element removal from the DOM using `remove()`

## Technologies Used

- HTML5
- JavaScript (ES6+)

## Running the Example

1. Open `index.html` in a web browser
2. Click the button to see the DOM manipulation in action
